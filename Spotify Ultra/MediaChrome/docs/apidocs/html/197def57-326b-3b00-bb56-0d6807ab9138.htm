Parameters
isolationLevel (IsolationLevel)
This parameter is ignored.
deferredLock (Boolean)
When TRUE, SQLite defers obtaining a write lock until a write operation is requested. When FALSE, a writelock is obtained immediately. The default is TRUE, but in a multi-threaded multi-writer environment, one may instead choose to lock the database immediately to avoid any possible writer deadlock.
Return Value
Returns a SQLiteTransaction object.
