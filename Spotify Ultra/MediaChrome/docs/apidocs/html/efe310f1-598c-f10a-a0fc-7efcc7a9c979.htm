The open flags to use when creating the connection
maxPoolSize (Int32)
The maximum size of the pool for the given filename
usePool (Boolean)
If true, the connection can be pulled from the connection pool
Remarks
Implementers should call SQLiteFunction.BindFunctions() and save the array after opening a connection to bind all attributed user-defined functions and collating sequences to the new connection.
