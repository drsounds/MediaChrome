BindFunctions Method (sqlbase)
Namespaces ► System.Data.SQLite ► SQLiteFunction ► BindFunctions(SQLiteBase)

[This is preliminary documentation and is subject to change.]

Called by SQLiteBase derived classes, this function binds all user-defined functions to a connection. It is done this way so that all user-defined functions will access the database using the same encoding scheme as the connection (UTF-8 or UTF-16).

Return Value
Returns an array of functions which the connection object should retain until the connection is closed.
Remarks
The wrapper functions that interop with SQLite will create a unique cookie value, which internally is a pointer to all the wrapped callback functions. The interop function uses it to map CDecl callbacks to StdCall callbacks.
