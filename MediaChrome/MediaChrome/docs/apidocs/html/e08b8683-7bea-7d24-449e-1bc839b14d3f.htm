The source connection preparing the command. Can be null for any caller except LINQ
strSql (String)
The SQL command text to prepare
previous (SQLiteStatement)
The previous statement in a multi-statement command, or null if no previous statement exists
timeoutMS (UInt32)
The timeout to wait before aborting the prepare
strRemain ( String %)
The remainder of the statement that was not processed. Each call to prepare parses the SQL up to to either the end of the text or to the first semi-colon delimiter. The remaining text is returned here for a subsequent call to Prepare() until all the text has been processed.
Return Value
Returns an initialized SQLiteStatement.
